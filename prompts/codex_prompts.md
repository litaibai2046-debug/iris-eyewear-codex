# Sample Codex Prompts for Iris Eyewear App

## Generate Scan Function
You are an expert Swift developer. Write a Swift function using ARKit that scans a user's face and returns a 3D mesh suitable for trying on virtual eyewear.

## Build UI for Try-On Screen
You are an expert iOS UI designer. Create a SwiftUI view that displays a 3D model of eyewear on top of the user's scanned face and allows the user to rotate and zoom the model.

## Suggest Frame Styles
As an assistant, output a list of three eyewear frame styles based on provided user preferences (e.g. round, square, lightweight) in JSON format.
